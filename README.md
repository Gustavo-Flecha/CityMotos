# CityMotos Child Theme

Child Theme personalizado para WooCommerce desarrollado para CityMotos, una tienda especializada en repuestos y accesorios para motocicletas.

## Objetivo

Este proyecto nació con el objetivo de adaptar WooCommerce a las necesidades operativas y comerciales de CityMotos mediante personalizaciones específicas de interfaz, navegación y experiencia de compra.

## Sitio en producción:

www.citymotos.com.ar


## Problema

La instalación estándar de WooCommerce no cubría varios requerimientos del negocio:

* Navegación rápida entre categorías de repuestos.
* Mejor experiencia de búsqueda para vendedores y clientes.
* Presentación visual adaptada al catálogo de motocicletas.
* Optimización del proceso de compra.

* Integración con herramientas desarrolladas específicamente para la operación diaria.

## Solución

Se desarrolló un Child Theme personalizado sobre WooCommerce incorporando funcionalidades y modificaciones orientadas a mejorar la experiencia de usuario y facilitar la gestión comercial.
