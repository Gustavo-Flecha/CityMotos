# CityMotos Child Theme

Child Theme personalizado para WooCommerce desarrollado para CityMotos, una tienda especializada en repuestos y accesorios para motocicletas.

## Objetivo

Este proyecto nació con el objetivo de adaptar WooCommerce a las necesidades operativas y comerciales de CityMotos mediante personalizaciones específicas de interfaz, navegación y experiencia de compra.

## Sitio en producción:

www.citymotos.com.ar


## Problema

La instalación estándar de WooCommerce no cubría varios requerimientos del negocio:

* Navegación rápida entre categorías de repuestos.
* Mejor experiencia de búsqueda para vendedores y clientes.
* Presentación visual adaptada al catálogo de motocicletas.
* Optimización del proceso de compra.
* Integración con herramientas desarrolladas específicamente para la operación diaria.

## Solución

Se desarrolló un Child Theme personalizado sobre WooCommerce incorporando funcionalidades y modificaciones orientadas a mejorar la experiencia de usuario y facilitar la gestión comercial.

## Funcionalidades Implementadas

### Navegación Personalizada

* Menú lateral para acceso rápido a categorías.
* Reorganización de elementos de navegación.
* Accesos directos a secciones importantes.

### Personalización WooCommerce

* Modificaciones de plantillas.
* Ajustes en páginas de producto.
* Mejoras en la experiencia de compra.
* Integración con funcionalidades personalizadas.

### Elementos Visuales

* Slider promocional.
* Hotspots interactivos.
* Componentes visuales orientados al catálogo de repuestos.

### Integración con Herramientas Internas

Este proyecto trabaja junto con desarrollos complementarios destinados a:

* Búsqueda por SKU.
* Búsqueda por código de barras.
* Gestión de stock.
* Procesos internos de venta.

## Tecnologías Utilizadas

* WordPress
* WooCommerce
* PHP
* HTML
* CSS
* JavaScript

## Arquitectura

El proyecto se implementa como Child Theme para mantener compatibilidad con futuras actualizaciones del tema principal y preservar las personalizaciones desarrolladas.

Las modificaciones se realizan utilizando:

* Hooks de WordPress
* Hooks de WooCommerce
* Plantillas personalizadas
* Assets CSS y JavaScript propios

## Caso Real

Este proyecto se encuentra basado en necesidades reales de operación comercial y fue desarrollado para acompañar la gestión diaria de CityMotos.

El objetivo principal fue reducir fricción en la navegación, facilitar el acceso a productos y mejorar la experiencia tanto para clientes como para vendedores.

## Estructura

- `/css/` → Archivos CSS separados por propósito
- `functions.php` → Carga de estilos
- `style.css` → Metadatos del tema + imports
- `aero.css` → Glassmorphism con backdrop-filter
