# Storefront Child Theme - Citymotos

Child theme basado en Storefront para un sistema WooCommerce personalizado con estilo visual Aero Glass, modular y profesional.

## Estructura

- `/css/` → Archivos CSS separados por propósito
- `functions.php` → Carga de estilos
- `style.css` → Metadatos del tema + imports
- `aero.css` → Glassmorphism con backdrop-filter

## Requisitos

- WordPress 6.8+
- Tema padre: Storefront
